# Security Policy

## Supported Versions

Only the latest code on the default branch is actively supported for security fixes.

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Instead, email:

- `contact@sahanpramuditha.com`

Include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation (if available)

You can expect an initial response within 72 hours.

## Disclosure Process

- Confirm and reproduce the report
- Prepare and validate a fix
- Release the fix
- Publish a coordinated disclosure summary
